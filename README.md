## Penjelasan

Repo ini terdiri dari dua program:  
**game_solver**: Menerima data mengenai sliding ice puzzle dan menyelesaikannya memakai A\* 

## Requirement

gcc jika ingin kompilasi.

## Cara Kompilasi

make

## Cara Menggunakan

**game_solver**  
1. Buka program
2. Input dimensi puzzle
3. Input data lain puzzle
4. pilih heuristik:
   - 0 : Manhattan
   - 1 : Cherbyshev
   - 2 : Euclidean
5. Pilih playback atau tidak
6. Navigasi playback dengan kontrol:
   - , : previous
   - . : next
   - e : goto specific step
   - q : quit
